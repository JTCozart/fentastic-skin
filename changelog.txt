[B]FENtastic Plus - Changelog[/B]

This is a community fork maintained by JTCozart, based on FENtastic Plus by
Ivar Brandt and Zaxxon709 (an Estuary MOD). Fork releases use major.minor.build
versioning.

[B]v1.0.16[/B]
- Power/system menu and select dialogs (e.g. PVR and favourites pickers) now
  match the context menu's look: a dimmed backdrop with floating buttons
  instead of a bordered panel.

[B]v1.0.15[/B]
- Added an optional Digital Release date for movies in the InfoPanel,
  sourced from the free MDbList API (same key used for ratings).
- Fixed the "Released" date in the InfoPanel showing only the year -
  it now shows the full mm/dd/yyyy date for both movies and TV shows.

[B]v1.0.14[/B]
- Fixed the release date still showing on the InfoWall and WideInfoWall
  poster views (v1.0.13 only removed it from the plain Poster views).

[B]v1.0.13[/B]
- Movie and TV posters no longer show the release date (still shown as the
  full date when you click into a title's details).
- TV shows always show a watched/total episode count, even "0/total" if
  unstarted.

[B]v1.0.12[/B]
- Install docs simplified to a single, reliable path: download the repository
  zip from the Releases page and use Kodi's "Install from zip file" directly.
  Dropped the File manager / repo-URL browsing option - Kodi's HTTP directory
  browsing and repository checksum fetch against it proved unreliable in
  testing (0-byte file sizes shown in File manager; repository checksum
  fetches timing out on some networks).
- TV show thumbnails now show watched/total episode count instead of
  unwatched/total.

[B]v1.0.10[/B]
- Publish script.fentastic.helper from this repository's Kodi repo, so
  installing via the repository (Option B) now auto-installs/updates the
  required helper add-on instead of needing a separate manual zip.

[B]v1.0.9[/B]
- Serve the Kodi repository from the GitHub Pages /docs folder so
  https://jtcozart.github.io/fentastic-skin/repo/ resolves correctly.
- Release pipeline reworked: the repo is generated into docs/repo and committed.

[B]v1.0.8[/B]
- Docs: point the required-helper link to the upstream download.
- Marketing page: add maintainer links (linktr.ee/jtczrt) to the footer.

[B]v1.0.7[/B]
- Docs: state the required, version-locked helper add-on
  (script.fentastic.helper 100.6.25g) that must be installed first.
- Landing page and README now identify the project as a fork and link upstream.
- Release pipeline hardened (GitHub Pages auto-configuration).

[B]v1.0.6[/B]
- TV shows: thumbnail episode badge now shows UNWATCHED / total instead of
  watched / total, across all view types.
- Merged latest upstream changes into the fork.
- Context menu, home, search results, and various dialogs: UI/UX refinements.
- Project housekeeping: new README, automated GitHub Actions release pipeline,
  Kodi repository add-on for auto-updates, and a project landing page.

[B]Upstream[/B]

FENtastic Plus

A modern, customizable Kodi skin - now with a proper release pipeline and one-click auto-updates.

Community fork of Zaxxon709/fentasticplus, maintained by JTCozart.

Features

Modern UI

A redesigned experience built on Kodi's Estuary skin.

Extra viewtypes

"WideWall" and "WideInfoWall" plus tuned built-in views.

Customizable widgets

Movie, TV-show and episode widgets, incl. stacked category widgets.

Custom main menu

Configurable movie and show menu items and paths.

Ratings & InfoPanel

Ratings for movies, shows, seasons and episodes, with an optional InfoPanel.

Multi-search

Search several categories at once, including Trakt lists.

Unwatched progress

TV thumbnails show unwatched / total episodes at a glance.

Auto-updates

Install the repository once and stay up to date automatically.

Designed for use with the FEN family of add-ons (Fen Light, Red Light, The Gears, Umbrella, POV, TMDbH).

Install

This skin depends on the script.fentastic.helper add-on. Installing the FENtastic Plus repository below resolves that dependency automatically and keeps everything updated afterward.
  1. Download repository.fentastic-x.x.x.zip from the Releases page onto the device running Kodi.
  2. In Kodi: Settings → Add-ons → Install from zip file, browse to where you saved it, and install it.
  3. Install from repository → FENtastic Plus Repository → Look and feel → Skin → FENtastic Plus → Install. Kodi resolves the script.fentastic.helper dependency from the same repository automatically - no separate helper install needed.

If Kodi blocks the install, enable Settings → System → Add-ons → Unknown sources (on some builds this is under Settings → Add-ons directly). Once installed, Kodi will keep both the skin and the helper updated automatically whenever a new version is released.
